Why This Works

  • Luma is already the source of truth. Events on the Claude Community calendar are legitimate by definition.
  • Access control lives where leaders already work. Adding GitHub IDs to a Luma description is the only new step.
  • Zero human review for verification. The Action validates against Luma. Humans only intervene on failure.
  • No org membership required. Fork, PR, and the Luma description is your access list.
